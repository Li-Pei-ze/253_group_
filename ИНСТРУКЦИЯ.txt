Зайдите в папку на локальном компьютере, в которую будет копироваться содержимое репозитория.

Запустите приложение Git Bash.

Скопируйте Git-репозиторий nlebedin в папку, например, MyDir командой:  

git clone https://github.com/nlebedin/253_group.git MyDir

 
Перейдите в папку MyDir с помощью команды cd имя_папки (cd от change directory):
 
cd MyDir   


Выполнив команду ls (ls от list), отображающую содержимое папки, убедитесь, что содержимое репозитория скопировалось:

ls


git branch  // посмотрите, какие есть ветки

git checkout -b forWork  // создайте на локальном компьютере ветку для работы forWork и перейдите в эту ветку

git branch   // снова смотрим, какие есть ветки

Далее положите новый файл в папку MyDir через проводник


git status 

git add test.cpp  // добавляем файл test.cpp

git commit -m 'Имя студента'  // сохраняем изменения

git checkout master   // переходим в ветку master на локальном компьютере
(через проводник увидим, что не будет файла test.cpp)


git merge forWork  // сливаем ветку forWork в основную ветку master    (заметим, что файл test.cpp опять появился)

git push origin master // отправим в удаленный Git-репозиторий

заходим в Git-репозиторий и проверяем изменения
